# TAPBOARD GAME

Pour lancer le projet il faut lancer plusieurs commandes : 

- [x] npm start dans le répertoire
- [x] sudo mongod pour lancer la base
- [x] ng build --output-path ../www/ --watch dans le répertoire du client Angular
